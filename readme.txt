RunApp v0.1 for Windows

介绍
    RunApp是一款通过运行对话框来启动应用程序的实用工具
    程序安装完成后，会在资源管理器(包括桌面)右键菜单的"发送到"菜单项中添加子项“命令快捷方式”
    凡发送到命令快捷方式的程序或其快捷方式，可通过运行对话框来启动

作者
    Fuwei Chin
    Email: fuweichin@gmail.com

许可证
    Apache 2.0

安装
    运行install.bat以安装程序

使用
    右击可执行文件或其快捷方式(如foo.exe)，发送到命令快捷方式
    按Win+R调出运行对话框，输入不带扩展名的程序名(不区分大小写，如foo)并确定，即可启动程序

卸载
    运行uninstall.bat以卸载程序
